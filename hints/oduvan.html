Como posso adicionar uma tupla à lista?

Para adicionar uma tupla a uma lista, você pode usar o método append. O exemplo a seguir mostra uma tupla que adiciona os múltiplos de três em sequência: 

termos = 4
multiplos = []
contador = 0
while contador < termos:
    multiplos.append((contador, 3 * contador))
    contador = contador+1
print(multiplos)  # Resultado: [(0, 0), (1, 3), (2, 6), (3, 9)]
